Reasearch Paper -> Retinal Disease Detection

This project explores the application of deep learning for automated detection of diabetic retinopathy (DR), a major cause of blindness globally, particularly among individuals with diabetes. Leveraging a publicly available dataset of 5,593 retinal fundus images, the system trains a Convolutional Neural Network (CNN) to classify DR into five severity levels. Key stages in the pipeline include: Image preprocessing using techniques like unsharp masking, contrast enhancement, and circle cropping Model training using CNNs for severity classification Evaluation based on precision, accuracy, F1-score, and recall The approach demonstrates strong potential for reliable, scalable DR screening, reducing dependence on manual screening and enhancing access to early diagnosis. Challenges such as class imbalance and dataset noise are addressed to improve robustness and generalization in future clinical applications.

Tech Stack: TensorFlow, Keras, OpenCV, Scikit-learn, Pandas, NumPy, Matplotlib

Core Skills: Image preprocessing, CNN-based classification, class balancing, medical AI
